Michelle Talley

Lecturer at the University of Central Arkansas

My professional interests include Computer Science Education and promoting and encouraging diversity in the computing field.

GitHub Campus Advisors are trained by GitHub experts on the best way to teach GitHub in a classroom. As a Campus Advisor for UCA since September 2019, I serve as an advocate and resource for GitHub on our campus. I also have GitHub goodies in my office, so stop by to grab some!

Girl Power in STEM (GPS) is a unique opportunity for 8th graders in Arkansas to gain hands-on experience and learn about the role women play in careers related to science, technology, engineering, and math. This event has been a personal passion since the conference was founded in 2014. We can always use volunteers, so let me know if you are interested.

Girls Who Code College Loop is a program that supports college-aged and early career women and non-binary people by helping them build belonging in tech and develop essential professional skills. Girls Who Code College Loop UCA is a Registered Student Organization! Please ask me if you are interested in participating.
